Обработка тегов подсветки синтаксиса

Для обработки моего Markdown-текста, Jekyll использует Liquid. Синтаксис следующий: 

{ % highlight lang (linenos)? % }
// code...
{ % endhighlight % }
Для начала, конечно, сниппет: 

<snippet>
    <content><![CDATA[
{ % highlight ${1:python} ${2}% }
${3}
{ % endhighlight % }
]]></content>
    <!-- Optional: Set a tabTrigger to define how to trigger the snippet -->
    <tabTrigger>highlight</tabTrigger>
    <!-- Optional: Set a scope to limit where the snippet will trigger -->
    <!-- <scope>source.markdown</scope> -->
</snippet>
А теперь о том, как просматривать написанный Markdown-текст. Джеффри советует воспользоваться программой Marked. Она очень удобная, красивая, поддерживает различные варианты отображения текста и т.п. Но одно она точно не определяет - теги подсветки синтаксиса, упомянутые выше.
